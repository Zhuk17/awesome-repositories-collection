- QQC[Supervision](https://github.com/roboflow/supervision)D** — 计算机视觉的通用Python框架,为现代CV项目提供可再使用的构件. 用于可视化,注释,对象计数,数据集管理,以及实时视频处理的模型-不可知性工具包.
-QQC[car-counter](https://github.com/serialdotai/car-counter)D**-开源Python工具,用于使用计算机视觉在交通视频中计数的自动车辆. 将YOLOv8对象检测与SORT跟踪算法结合,用于精确的车辆跨虚拟行计数.
